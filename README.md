## 萨卡兹肉鸽伤害计算器

萨卡兹肉鸽伤害计算器，是为明日方舟集成战略“萨卡兹的无终奇语”主题所开发的，简便道中与结局伤害计算的小型应用。

### 开发

本项目使用 **[Tauri v2](https://v2.tauri.app/)**. 请确保你安装了 [node >=22](https://nodejs.org/) 和 [Rust](https://www.rust-lang.org/learn/get-started).

```bash
git clone git@github.com:SundayChen/Arknights_Sarkaz_Rogue_Damage_Calculator.git
npm install
```

以开发模式运行本项目：

```bash
npm run tauri dev
```
